# Lead Any Mob
You will be able to attach leads to all kinds of creatures, such as villagers, zombies, and bats.

![ScreenShot](./images/screenshot.png)

## Permission
- `leadanymob.use`
  - Allow use of this plugin
  - All players have this permission by default
